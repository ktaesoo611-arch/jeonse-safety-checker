# 건축물대장 API 문의 - API 호출 시 totalCount 0 반환 문제

## 문의 제목
건축물대장HUB 서비스 API 호출 시 정상 응답(200)이나 데이터가 조회되지 않는 문제

## 문의 내용

### 1. 기본 정보
- **서비스명**: 국토교통부_건축HUB_건축물대장정보 서비스
- **API 엔드포인트**: `https://apis.data.go.kr/1613000/BldRgstHubService`
- **일반 인증키**: `1043939fd3528b974a27dd0c1707e2947060af8299d817929525e809a412ccca`
- **활용기간**: 2025-11-10 ~ 2027-11-10
- **신청유형**: 운영계정 | 활용신청
- **처리상태**: 승인

### 2. 문제 상황
API 호출 시 HTTP 200 응답을 받으나, 모든 주소에 대해 `totalCount: 0`이 반환되어 실제 데이터를 조회할 수 없습니다.

### 3. 테스트 내용

#### 테스트한 엔드포인트
1. `getBrRecapTitleInfo` (건축물대장 총괄표제부)
2. `getBrTitleInfo` (건축물대장 표제부)

#### 테스트한 주소 예시
```
1. 서울특별시 강남구 역삼동 737
   - sigunguCd: 11680
   - bjdongCd: 10600
   - platGbCd: 0
   - bun: 737
   - ji: 0

2. 서울특별시 용산구 이촌동 302-12
   - sigunguCd: 11170
   - bjdongCd: 10200
   - platGbCd: 0
   - bun: 302
   - ji: 12

3. 서울특별시 서초구 서초동 1305-5
   - sigunguCd: 11650
   - bjdongCd: 10800
   - platGbCd: 0
   - bun: 1305
   - ji: 5
```

#### API 호출 예시
```
GET https://apis.data.go.kr/1613000/BldRgstHubService/getBrTitleInfo
?serviceKey=1043939fd3528b974a27dd0c1707e2947060af8299d817929525e809a412ccca
&sigunguCd=11680
&bjdongCd=10600
&platGbCd=0
&bun=737
&ji=0
&numOfRows=10
&pageNo=1
```

#### 응답 내용
```json
{
  "header": {
    "resultCode": "00",
    "resultMsg": "NORMAL SERVICE"
  },
  "body": {
    "items": {
      "item": []
    },
    "numOfRows": "10",
    "pageNo": "1",
    "totalCount": "0"
  }
}
```

### 4. 확인 사항
- ✅ API 키 발급 완료
- ✅ 활용신청 승인 완료 (상세설명 페이지 확인)
- ✅ HTTP 403 Forbidden 에러 없음 (이전에는 발생했으나 현재는 해결됨)
- ✅ HTTP 200 정상 응답 수신
- ✅ resultCode "00", resultMsg "NORMAL SERVICE" 확인
- ✅ 필수 파라미터 모두 포함 (sigunguCd, bjdongCd, platGbCd, bun, ji)
- ✅ 서울시 법정동코드 및 시군구코드 정확히 입력
- ✅ 다양한 주소로 테스트 (강남구, 용산구, 서초구 등)

### 5. 질문 사항

1. **API 서비스 활성화 확인**
   - 현재 해당 API 서비스가 정상적으로 활성화되어 있는지 확인 부탁드립니다.
   - 활용신청 승인 후 추가 승인 절차나 대기 시간이 필요한지 문의드립니다.

2. **데이터 조회 불가 원인**
   - 파라미터는 정확하게 입력했으나 데이터가 조회되지 않는 이유를 알고 싶습니다.
   - 특정 지역이나 건물만 조회 가능한 제한사항이 있는지 확인 부탁드립니다.

3. **추가 조치 필요 여부**
   - API 사용을 위해 추가로 필요한 승인이나 설정이 있는지 확인 부탁드립니다.
   - 개발계정과 운영계정의 차이가 있는지, 별도 신청이 필요한지 문의드립니다.
